Hướng dẫn sử dụng chức năng ghép file tuỳ ý

  1. Tạo file key bản quyền bằng tab Tạo Key
  2. Chạy MobaXterm với tham số -customizer:
    .\MobaXterm.exe -customizer
  3. Xuất file tuỳ ý của bạn thành MobaXterm customization.custom
  4. Tải lên cả file key và file tuỳ ý bên dưới
  5. Bấm "Ghép file" để tạo file kích hoạt tổng hợp
